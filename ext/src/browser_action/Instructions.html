Step 1 -> Install Wampp Server into your System. Wampp can be downloaded from here ( 64 bit / 32 bit ) .
Step 2 -> Move "ext/src/PHP" folder to WWW folder of WAMPP which is likely "C:\wamp\www\PHP" .
Step 3 -> Click on Create Database Button to create database.
Step 4 -> Click on Fetch Data button to fetch data from college server.
Step 5 -> Click on Calculate Percentile button to calculate percentile of each student.
